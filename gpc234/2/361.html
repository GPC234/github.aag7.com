PC端微信多开方法
首先第一步，点击我们安装好的微信，右键属性，查找到我们安装的初始位置。我的初始安装目录是
D:\0\Tencent\WeChat\WeChat.exe
点击我们安装好的微信，右键属性-快捷方式-目标(T)
打开记事本，输入以下脚本：
start D:\0\Tencent\WeChat\WeChat.exe
start D:\0\Tencent\WeChat\WeChat.exe
注意想开几个微信，就把start D:\0\Tencent\WeChat\WeChat.exe复制几行
然后保存-保存的txt文档后缀名改为.bat
start D:\0\Tencent\WeChat\WeChat.exe
start D:\0\Tencent\WeChat\WeChat.exe
